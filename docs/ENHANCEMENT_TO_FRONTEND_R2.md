在左侧侧边栏增加一个新的入口“探索”，并开发一个适用于javdb的浏览器插件，能够在网站里完成多种功能，和该项目融合

探索会提供一个浏览器的入口，访问javdb.com。他可以实现多种功能：

1. 可以让用户在这里完成登录操作，然后直接从这里获取到cookie来更新config.py
2. 提供地址栏让用户可以直接看到url，同时在地址栏旁边直接增加一个“跳转至adhoc任务创建”，使用跳转时的url，并让user更新其他设置之后直接创建任务 （仅在除了电影详细页面之外的url中使用，仅在javdb中使用）
3. 在电影详细页面中，可以在每个种子后面增加一个按钮“使用qbittorrent下载”
4. 在电影详细页面中，可以增加按钮“一键下载”来下载最优组合的种子
5. 在索引页中，对每个电影都可以增加按钮“一键下载”，同时，在每一页都可以增加一键下载来下载整页的电影的种子
6. 在索引页，可以对每个电影旁边都增加一个小标签，标明是否有无码的种子（需要解析电影详细页面），是否已经下载过
